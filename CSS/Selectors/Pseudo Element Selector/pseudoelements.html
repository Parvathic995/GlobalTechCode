Pseudo Elements

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s). For example, ::first-line can be used to change the font of the first line of a paragraph. /* The first line of every paragrapgh element. */ p::first-line { color: blue; text-transform: uppercase; }
